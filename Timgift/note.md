The e-commerce website will feature a clean, user-friendly interface with the following structure:

1. Homepage: Displays featured products, categories, and a search bar for easy navigation.
2. Product Listings:Two sections— In-Stock and Pre-Order —showcasing products with images, descriptions, prices, and availability status.
3. Product Page: Details for each product, including a quantity selector that auto-calculates the total cost.
4. Checkout: Users input order details, review the auto-generated invoice, and submit the order, which is sent to the store WhatsApp number for processing.
5. Backend: Admin panel for managing stock, pre-orders, and viewing submitted orders.
